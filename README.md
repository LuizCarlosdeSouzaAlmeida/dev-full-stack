# Sistema de gerenciamento de curso Web

Projeto para disciplina de desenvolvimento Full Stack UFG 2023/1

## Front End

- React
- Mui
- Javascrypt
- Next
- yarn

Para rodar o projeto basta entrar na pasta e rodar os seguintes comandos:

```bash
    # Instala o projeto
    yarn
    # Inicia ambiente de Dev
    yarn dev
    
```

## Back End

- Kotlin
- Spring bot
- JPA
- REST
- Postgresql

Para rodar o projeto basta entrar na pasta e rodar os seguintes comandos:

```bash
    # Builda projeto
    ./gradlew build
    # Roda o projeto
    ./gradlew bootRun
```
